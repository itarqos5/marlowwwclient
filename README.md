## Compile Yourself (Avoid Pre-compiled .jars)

Firstly, clone the repo.
```
git clone https://github.com/nxghtCry0/marlowwwclient.git
```

Assuming you already have Eclispe Adoptium JDK 21+ installed:

Test the client in a developer environment:
```
gradlew runClient
```

Build the client for release or use the jar yourself:
```
gradlew build
```

This will produce a .jar file in the `build/libs/` directory.
You are looking for the `imnotcheatingyouare-1.jar` file, NOT the sources file.
